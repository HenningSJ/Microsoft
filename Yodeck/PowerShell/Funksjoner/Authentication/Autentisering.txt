Dette skriptet skal:

1. Validere at token finnes
2. Bygge headers korrekt
3. Håndtere GET / POST / PATCH / DELETE
4. (Valgfritt) serialisere body til JSON
5. Skjule REST‑detaljene for resten av koden

Jeg har laget et API Token under Account Settings i portalen.
Henning-Read

Lagret tokenet i HKEY_Current_User\Environment med dette skriptet:

[Environment]::SetEnvironmentVariable(
  "YODECK_TOKEN",
  "FYLLINNTOKENHER",
  "User"
)

Testet med denne kommandoen: 
$env:YODECK_TOKEN

Man kan også autentisere med denne:

$headers = @{
  Authorization = "Token YODECK_TOKEN:$($env:YODECK_TOKEN)"
}

Hvert kall må inneholde:
hvem du er (Authorization)
hva du vil gjøre (GET/POST/PATCH)
hvilken ressurs (URL)
Ettersom at serveren som mottar hver forespørsel har glemt deg med en gang.
